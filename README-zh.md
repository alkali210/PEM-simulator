# PEM 电解水实验仿真系统 - 程序使用说明 (Fluent UI 版)

## 1. 项目简介与功能概览
本程序是一款基于 **PySide6** 结合 **Fluent UI (qfluentwidgets)** 开发的 PEM（质子交换膜）电解水实验仿真系统。系统旨在通过现代化 UI 界面模拟电解水过程中的电化学特性。

### 主要功能
- **Fluent UI 交互体验**：采用微软 Fluent 设计语言。
- **自适应主题**：默认采用 `Theme.AUTO` 策略，自动匹配 Windows 系统深色/浅色模式。
- **实时仿真**：动态模拟电解水过程中的 V-I（电压-电流）响应。
- **参数调控**：支持设定操作电压、目标电流密度及环境温度。
- **可视化监控**：实时绘制极化曲线，并提供系统运行日志。
- **性能分析**：自动计算瞬时功率、电压效率及关键指标汇总。

---

## 2. 环境依赖与安装命令
本程序运行于 Python 3.x 环境，需安装以下第三方库：

### 核心依赖
- **PySide6**: UI 基础框架。
- **PySide6-Fluent-Widgets**: 提供 Fluent UI 组件（在代码中通过 `import qfluentwidgets` 使用）。
- **pyqtgraph**: 科学绘图，用于极化曲线显示。
- **numpy**: 仿真模型数值计算。
- **pytest/ruff/mypy**: 开发、测试与静态检查工具。

### 环境适应性
程序具备 Fluent UI 探测机制。若当前环境未安装 `PySide6-Fluent-Widgets`，系统将通过 `fluent_compat.py` 自动降级至标准 Qt (Fusion 风格) 运行，确保核心仿真功能在任何环境下均可启动。

### 安装命令
在终端（CMD 或 PowerShell）中运行以下命令：
```bash
pip install PySide6 PySide6-Fluent-Widgets pyqtgraph numpy pytest mypy ruff
```

## 3. 启动方式
在项目根目录下，执行以下命令即可启动 GUI 界面：
```bash
python src/main.py
```

## 4. 界面分区说明 (Fluent 版)
主界面采用 Fluent `CardWidget` 卡片式布局，分为以下五个核心区域：

1.  **左侧功能列** (占比 25%)：
    - **实验信息 (顶部)**：显示实验名称、当前运行状态以及累计运行时间。
    - **参数设置 (中上)**：采用 `DoubleSpinBox` 控件设定电压、电流密度和温度。
    - **运行控制 (中下)**：提供状态流转保护的仿真控制按钮。
    - **输出结果 (底部)**：实时计算并显示功率、电压效率及运行汇总指标。
2.  **右侧监控区** (占比 75%)：
    - **数值显示**：显示当前的电压、电流和温度实时值。
    - **V-I 极化曲线**：基于 `pyqtgraph` 动态绘制，支持主题联动（自动切换深/浅背景）。
    - **系统日志**：记录仿真过程中的关键状态切换。

---

## 5. 主题支持说明
本系统原生支持深色模式。切换方法：
- **自动适配**：程序启动时检测系统设置，自动切换主题。
- **代码强制**：可在 `src/widgets/fluent_compat.py` 中修改 `apply_app_theme` 调用参数。
- **图表兼容**：`pyqtgraph` 绘图层已通过 `isDarkTheme` 接口实现色彩联动，确保在深色背景下坐标轴清晰可见。

## 6. 典型操作流程
1.  **参数配置**：在“参数设置”区调整电压、电流密度和温度至所需范围。
2.  **启动仿真**：点击“启动”按钮，仿真开始，计时器启动，极化曲线开始绘制。
3.  **观察数据**：实时查看左侧“输出结果”和右侧“过程监视”区的变化。
4.  **中途调整**：
    - 点击“暂停”可冻结当前状态。
    - 点击“继续”恢复仿真运行。
5.  **完成与重置**：
    - 仿真运行 7 秒后会自动停止（或手动点击“停止”）。
    - 点击“重置”将清除所有历史曲线、计时器和汇总数据，恢复至初始状态。

---

## 7. 参数含义与建议范围
| 参数 | 符号 | 物理意义 | 建议范围 |
| :--- | :--- | :--- | :--- |
| **电压** | $V$ | 加在电解槽两端的电势差 | 1.23 - 3.0 V |
| **电流密度** | $j$ | 单位膜面积通过的电流 | 0.0 - 5.0 A/cm² |
| **温度** | $T$ | 电解槽的工作温度 | 20.0 - 90.0 °C |

*注：电解槽有效膜面积固定为 4.0 cm²，系统会自动计算总电流 $I = j \times Area$。*

## 8. 输出结果说明
- **瞬时功率**：反映电解槽当前的能量消耗速率。
- **电压效率**：$\eta_V = 1.481 / V_{actual}$。当电压接近热中性电压时，效率较高。
- **极化曲线**：V-I 曲线反映了电解槽的内阻特性，曲线斜率越小通常意味着欧姆损耗越低。
- **系统日志**：详细记录了每个状态切换的时间戳，便于回溯实验过程。

## 9. 常见问题与排查
- **无法启动程序**：请检查是否已正确安装 PySide6。运行 `pip list` 确认。
- **曲线不显示**：确保点击了“启动”按钮，且“电压”设置在 1.23V 以上（低于理论分解电压将无显著电流）。
- **界面显示不全**：建议在 1080p 或更高分辨率下运行，或手动拉伸主窗口。
